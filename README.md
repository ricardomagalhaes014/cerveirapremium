# Cerveira Premium — Landing page

Site de apresentação do empreendimento **Cerveira Premium** (lote 4, Vila Nova de Cerveira), com a **dps Imobiliário**.

Site estático (HTML + CSS + JS, sem build). Pensado para publicar em:

```
https://dpsimobiliario.pt/cerveirapremium
```

## Estrutura

```
cerveirapremium/
├── index.html
├── assets/
│   ├── css/style.css
│   ├── js/main.js
│   ├── logo-dps.svg
│   └── img/            (renders do empreendimento)
├── robots.txt
└── README.md
```

Caminhos relativos — funciona em subpasta (`/cerveirapremium`) ou domínio próprio.
